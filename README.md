# 🍅 番茄时钟（Vue3+Vite 项目）

## 🧩 项目简介
本项目是一个基于 Vue 3 + Vite + Vant + Pinia 构建的专注番茄时钟工具。支持明暗模式、音效提醒、进度环倒计时、移动端自适应等功能。

## ✨ 实现功能
- 25分钟番茄专注计时器
- 开始 / 暂停 / 重置操作
- 倒计时结束 Toast 提示 + 音效提醒
- 进度圆环动态显示
- 明暗模式切换（支持本地保存）
- 音效开关（本地保存）
- 完全移动端适配
